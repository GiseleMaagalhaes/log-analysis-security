# 🔎 Log Analysis Security

Projeto em Python para análise de logs de acesso.

## 📌 Objetivo
Identificar:
- Tentativas de login com falha
- IPs suspeitos
- Padrões de acesso

## 🚀 Tecnologias
- Python

## ▶️ Como executar
Execute:
python log_analysis.py

## 📊 O que o projeto faz
- Conta falhas de login
- Mostra IPs mais ativos
- Detecta comportamento suspeito

## 💡 Exemplo
Detecta IPs com muitas tentativas de acesso
